# 第四章 Rails 中的模型（Model）

## 课程概要：

本课程讲解 Model 中基本的 CRUD 操作，使用 migrate 命令迁移数据库，以及如何避免常见的查询问题。

## 知识点：

1. CRUD
2. 数据库迁移（Migration）
3. 表间关联（Relations）
4. 属性校验（Validates）
5. 回调（Callback）
6. Rspec 测试

## 课程背景

模型（Model）是 MVC 架构中的 M，代表数据库。Rails 对数据库的操作都是在模型中完成的，对模型的学习，可以了解 Rails 如何实现数据库表间的关联关系，以及 Rails 中如何优化数据库查询。
